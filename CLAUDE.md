# 魔法女巫网站 - CLAUDE.md

## 项目简介
「灵汐之门」是一个面向12-45岁女性、主题为魔法女巫的纯静态网站。包含女巫类型测试、魔杖定制、塔罗占卜、能量壁纸四大功能。部署于Gitee Pages，零预算，全中文。

## 核心约束
- **零预算**：不使用任何付费服务/API/字体/图片
- **纯静态**：HTML + CSS + JavaScript，无后端
- **全中文**：所有界面和内容使用简体中文
- **响应式**：手机/平板/电脑均需适配
- **暗黑神秘风**：整体视觉风格关键词——星空、月亮、暗色系、神秘感

## 工作约定
- 先确认需求，再编写代码
- 内容设计优先于视觉实现
- 每完成一个阶段，更新执行步骤和开发日志
- 涉及设计决策时，向用户确认后再执行
- 代码不加注释，除非逻辑非常不直观
- 使用现代浏览器API，不兼容IE

## 目标用户画像
- 12-45岁女性
- 对神秘学、女巫文化、魔法、塔罗牌、占卜、能量疗愈感兴趣
- 可能是初学者或有一定了解的爱好者
- 使用手机为主要访问设备

- 温和、细腻、有力量感
- 给用户想象空间，不做绝对化判断
- 正面积极，赋权予用户
