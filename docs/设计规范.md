## 字体

- **标题**：使用系统衬线体（Georgia / "Noto Serif SC" / "Songti SC"），优雅纤细，有杂志感
- **正文**：系统无衬线体（"PingFang SC" / "Microsoft YaHei" / -apple-system），保证可读性
- **「灵汐之门」logo**：衬线体 + 较大字号 + 月光金色 + 字母间距
- 手机正文最小16px

## 首页体验

**仪式感渐进式**：
1. 深色背景 + 缓慢闪烁的星空粒子
2. 一轮月亮（或微光）从画面中央浮现
3. 「灵汐之门」四字缓缓淡入，短暂静默
4. 三大模块入口（测试/塔罗/壁纸）像星火逐一亮起
5. 用户自然向下探索

## 装饰元素

**天文 + 魔法融合**：
- 星空/星轨/星辰粒子 → 背景氛围层
- 月相变化图（满月/新月/月牙）→ 装饰性图案
- 六芒星/五芒星 → 功能图标
- 水晶球、草药剪影 → 模块装饰
- 四大元素三角符号 → 女巫类型标识

## 塔罗牌视觉

**初期方案**：CSS绘制
- 深色卡牌底（`#16213E`）+ 月光金边框
- 牌面中央显示符号/星象图案（CSS/SVG）
- 牌名用衬线体居中
- 背面向日葵星芒几何图案（深紫底 + 金色线条）
- 抽牌时有翻转动画（3D transform）

**后续可升级为原创插画**

## 布局规范
- 最大内容宽度：480px（手机）/ 768px（平板）/ 1200px（桌面）
- 卡片圆角：12px
- 按钮圆角：24px（胶囊形）
- 内边距：16px（手机）/ 24px（桌面）
- 模块间距：24px

## 动效
- 首页渐进：星星闪烁 → 月光浮现 → 标题淡入 → 入口亮起（总时长约4-5秒）
- 页面切换：淡入淡出，300ms
- 选项选中：轻微缩放 + 发光边框（对应元素色）
- 卡牌翻转：3D transform，600ms
- 结果展示：逐条浮现（staggered animation）
- 星星背景：缓慢闪烁循环，随机节奏
- 按钮悬停：微光呼吸效果
